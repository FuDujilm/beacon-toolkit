# Ham Calculator Hub Requirements

## 背景

当前无线电工具页已经有若干独立计算器，但“计算”类别仍然偏散。用户要求参考 `hamradiobase` 的计算器目录，把该网站出现的计算器纳入当前应用的信息架构，并额外新增一个“快速计算”类别。

## 用户故事

- 作为用户，我希望在“工具”页的“计算”分类里看到更完整的计算器目录，而不是零散几个入口，这样我能快速判断当前版本有哪些工具。
- 作为用户，我希望新增“快速计算”类别，把最常用、最低门槛的射频计算集中起来，减少跳转层级。
- 作为用户，我希望已完成的计算器能直接进入可用页面，未完成的条目也要有清晰入口和说明，而不是消失。

## 范围

### 包含

- 在无线电工具页新增“快速计算”分类。
- 新增“计算目录”页面，参考 `hamradiobase` 的九大类：
  - 天线
  - 传输线
  - 传播
  - 电子
  - 阻抗匹配
  - 数字模式
  - VHF / 卫星
  - 日志 / 竞赛
  - 电源 / 便携
- 分类页内展示对应计算器条目，已实现项可直接进入已有页面，未实现项使用明确占位说明。
- 新增“快速计算”页面，至少提供：
  - 欧姆定律
  - 瓦特 / dBm / dBW
  - SWR / 回波损耗
- 新增“传输线计算器”页面，至少提供：
  - 同轴匹配损耗
  - SWR / 回波损耗 / 反射系数
  - 电气长度与物理长度换算
  - 四分之一波阻抗变换

### 不包含

- 本次不实现 `hamradiobase` 上全部交互计算逻辑。
- 不引入外部网页嵌入或 WebView。
- 不变更后端接口或新增网络依赖。

## 验收标准

- 工具页侧边分类包含“快速计算”。
- “计算”分类下存在“计算目录”入口。
- “快速计算”分类下存在“快速计算”入口。
- “计算目录”能看到九大类及其代表性计算器条目。
- 已实现条目至少可直接跳到：
  - 单位换算
  - 时区计算
  - QTH 定位
  - 八木天线计算器
  - 镜像频率计算器
  - 互调计算器
  - 卫星追踪
- 快速计算页支持输入互算，并保留免责声明。
- 传输线条目不再全部是占位；至少有一个统一入口可完成上述四类计算。
